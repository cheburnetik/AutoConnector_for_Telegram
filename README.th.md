# AutoConnector for Telegram

AutoConnector for Telegram — แอปจะค้นหาพร็อกซี MTProto บนอินเทอร์เน็ตด้วยตัวเอง ตรวจสอบว่ายังใช้งานได้อยู่ และตั้งรีเลย์ในเครื่อง ซึ่งทำให้ Telegram ทำงานได้อย่างเสถียรแม้ในที่ที่มันถูกบล็อก ผู้ใช้ไม่จำเป็นต้องค้นหาพร็อกซีที่ใช้งานได้ด้วยตนเอง — AutoConnector for Telegram จะเลือกพร็อกซีที่เร็วที่สุดและยังใช้งานได้อยู่ตลอดเวลา โดยทดสอบจากคอมพิวเตอร์/โทรศัพท์ของคุณโดยตรง

พูดอีกอย่างคือ มันคือเครื่องสแกนช่อง Telegram และการสมัครรับข้อมูลทุกประเภทที่มีพร็อกซี MTProto สาธารณะฟรี พร้อมเสียบเข้า Telegram ของคุณโดยอัตโนมัติ ไม่ต้องอัปเดตไคลเอนต์ Telegram การตรวจสอบความพร้อมใช้งานของพร็อกซีทำจากอุปกรณ์และสภาพแวดล้อมเครือข่ายของคุณโดยตรง ทำงานบน WiFi+LTE โดยไม่ต้องใช้ VPN

แพลตฟอร์ม: Android, Windows, Linux, MacBook

เวอร์ชัน Android ทำงานได้ด้วยตัวเอง ส่วนใน Windows คุณต้องตั้งเอนจินพร็อกซี "СПЛИТ..." (Split) หรือ "КОАЛЕСИНГ.." (Coalescing) — ในการตั้งค่าหรือปุ่มที่อยู่ทางขวาของคำว่า "Telegram เชื่อมต่อแล้ว" และวงกลมสีเทา/เขียววงใหญ่ หรือเลือกตัวที่ทำงานได้ดีที่สุดสำหรับคุณเป็นการส่วนตัว เพราะการบล็อกในแต่ละที่ต่างกัน โหมด "КОАЛЕСИНГ.." (Coalescing) ใช้เป็นทางเลือกสุดท้าย — Telegram จะเริ่มทำงาน แต่การส่ง/แสดงเนื้อหามีเดียในแชทจะเสียไป

ขอแนะนำให้ปิดไฟร์วอลล์อย่าง COMODO หากคุณใช้งานอยู่ มันจะดันแอปเข้าแซนด์บ็อกซ์ และไฟร์วอลล์ของมันจะทำให้การเชื่อมต่อขาออกไปยังพร็อกซี MTProto เสียไป หรือรันแอปในเครื่องเสมือน ที่นั่น TCP stack จะเปลี่ยนไปทั้งหมด พฤติกรรมของ AutoConnector จะแตกต่างออกไป

และยังช่วย Telegram สลับระหว่างพอร์ต 55001 และ 55002 ด้วยมือ (ในการตั้งค่าพร็อกซี) เพื่อให้เข้าถึงพร็อกซีที่ใช้งานได้และข้อความ "เชื่อมต่อแล้ว" ภายใน Telegram ได้เร็วขึ้น

# ✨ ความสามารถ

- **ค้นหาพร็อกซีอัตโนมัติ** — สแกนหน้าเว็บและการสมัครรับข้อมูลแบบเปิดหลายสิบรายการ
- **ตรวจสอบว่ายังใช้งานได้** — MTProto-handshake จริง จัดอันดับตามความเร็ว/ความเสถียร
- **รีเลย์ในเครื่อง** — Telegram เชื่อมต่อกับ `127.0.0.1` ส่วน AutoConnector for
  Telegram จะกำหนดเส้นทางการรับส่งข้อมูลผ่านพร็อกซีที่ดีที่สุดและยังใช้งานได้ และสลับ
  หากตัวปัจจุบันล่ม
- **ต่อต้าน DPI** — ชุดเทคนิคพรางตัว (เลียนแบบเบราว์เซอร์ การแบ่งย่อย
  แพ็กเก็ต FakeTLS และอื่น ๆ) โหมด «ไล่ลองอัตโนมัติ» จะเลือกตัวที่ใช้งานได้ด้วยตัวเอง
- **สถิติแบบละเอียด** — พร็อกซีที่ใช้งานได้/ใช้ไม่ได้ ความเร็ว ความหน่วง การรับส่งข้อมูล
  ประสิทธิภาพของแต่ละเทคนิคต่อต้าน DPI
- **แคตตาล็อกพร็อกซี** — อันดับสูงสุดตามคะแนน พร้อมการ์ดรายละเอียดของแต่ละโฮสต์
  สำหรับแต่ละโฮสต์จะเห็น «การเชื่อมต่อ Telegram / ที่สำเร็จ / การตรวจสอบทั้งหมด» และ
  ประวัติ 25 ครั้งล่าสุดของการพยายาม (ระยะเวลาเชื่อมต่อ TCP/TLS/รวม รับ/ส่งไบต์)
- **การเลือกโฮสต์ที่ยืดหยุ่น** — แถบเลื่อน «ความกว้าง»: ตั้งแต่ «ยึดตัวที่ผ่านการพิสูจน์ดีที่สุด»
  ไปจนถึง «ลองพร็อกซีที่ใช้งานได้หลากหลายให้กว้างที่สุด» เมื่อ Telegram วิ่งวุ่นไปมาตามพอร์ต
  ของรีเลย์ การเลือกจะขยายโดยอัตโนมัติ แถบเลื่อนแยกต่างหาก — ไทม์เอาต์
  ของการเชื่อมต่อ (100 มิลลิวินาที … 15 วินาที) และ «การแข่งขันของ upstream» (เชื่อมต่อหลายตัวพร้อมกัน)
- **12 ภาษา** ของอินเทอร์เฟซพร้อมการตรวจหาอัตโนมัติ รองรับ RTL

# เมื่อเปิดใช้งานครั้งแรก สิ่งสำคัญคือ:

- ตั้งค่า Telegram โดยระบุพร็อกซี SOCKS5 แบบตายตัว localhost:55001 และ localhost:55002
- ลบพร็อกซีอื่น ๆ ออก ยกเว้นสองตัวนี้
- เปิด "ใช้พร็อกซี" ใน Telegram
- อย่าบล็อกการแจ้งเตือนบน Android มิฉะนั้นจะไม่ทำงานในเบื้องหลัง
- เมื่อเปิดใช้งานครั้งแรก ให้รอ ~15 นาที จนกว่าจะดาวน์โหลดและไล่ลองพร็อกซี MTProto เสร็จ และจนกว่าไคลเอนต์ Telegram เองจะสลับ

# หากไม่มีอะไรทำงานเลย

1. หากไม่มีอะไรทำงานเลย ให้ใช้โปรแกรมเป็นแคตตาล็อกพร็อกซีที่ใช้งานได้แบบอัตโนมัติ ด้วยฮอตคีย์ CTRL+WIN+ALT+P และ CTRL+SHIFT+ALT+P คุณสามารถเพิ่มพร็อกซีแบบสุ่มเข้า Telegram ของคุณได้อย่างรวดเร็วโดยไม่ต้องเปิดหน้าต่าง AutoConnector ผลที่ได้คือ Telegram จะไปยังพร็อกซีโดยตรง โดยไม่ผ่าน AutoConnector แต่คุณไม่จำเป็นต้องเฝ้าดูแชทที่มีการเผยแพร่พร็อกซีฟรีและเสียเวลาตรวจสอบมัน ปล่อยให้ AutoConnector อยู่ในเทรย์ ปิดสวิตช์ "Connector" เปิดสวิตช์ "Scan" ไว้

2. ลอง AutoConnector บนอุปกรณ์อื่น: โทรศัพท์ โทรศัพท์ของเพื่อน คอมพิวเตอร์ บนแพลตฟอร์ม Windows/Android หลักการหลบหลีกการบล็อกต่างกันโดยสิ้นเชิง เป็นไปได้สูงว่าบน Android จะทำงานได้โดยไม่ต้องตั้งค่าใด ๆ

3. หากไม่มีอะไรทำงานเลย ให้เปิด VPN ชั่วคราวสักหนึ่งวัน และทดสอบ AutoConnector + Telegram ในวันนั้น (เชื่อมต่อ Telegram ผ่าน AutoConnector โดยระบุพร็อกซีบนพอร์ต 55001 และ 55002) ภายในโปรแกรมให้เปิดเครื่องหมายถูก "เมื่อเปิด VPN -> ส่งผ่านพร็อกซีไปยัง MTProto" ดูสิ AutoConnector เริ่มทำงานหรือไม่? หากใช่ ข้อสรุปก็ชัดเจน — AutoConnector ค้นหาพร็อกซีได้สำเร็จ ส่งต่อการรับส่งข้อมูลของ Telegram ไปที่นั่นได้สำเร็จ แต่หากปิด VPN ระบบบล็อกในประเทศของคุณบล็อกการเชื่อมต่อขาออกทั้งหมดอย่างเข้มงวดเกินไป ในกรณีนั้นคุณจะต้องใช้เวลาราวครึ่งชั่วโมงไล่ลองเครื่องมือใน AutoConnector เพื่อเลือกตัวที่ใช้งานได้ ซึ่งสามารถหลบหลีกการบล็อกได้ โปรแกรมยังไม่สามารถไล่ลองทุกตัวเลือกแบบอัตโนมัติทั้งหมดได้ (มีเพียงการไล่ลองเทคนิค anti-DPI อัตโนมัติเท่านั้น) หลังการทดลองให้ปิด VPN และคืนค่าเครื่องหมายถูก "เมื่อเปิด VPN ส่งผ่านพร็อกซีไปยัง MTProto" กลับเป็นสถานะ "ส่งผ่านโดยตรง"

4. ทางเลือกแทนข้อ (3) ติดตั้งเครื่องเสมือนบนโฮสต์ Windows/Linux ของคุณ ในนั้นรัน Telegram + AutoConnector มันเริ่มทำงานได้อย่างสมบูรณ์แบบแม้ไม่มี VPN? แสดงว่าโฮสต์ของคุณเองที่ทำให้การเชื่อมต่อของคุณเสีย ไม่ใช่ระบบบล็อกในประเทศของคุณ! สาเหตุ: ไฟร์วอลล์ แอนตี้ไวรัส เศษซากจาก VPN หากแอนตี้ไวรัสนำ AutoConnector เข้าแซนด์บ็อกซ์ หรือไฟร์วอลล์บล็อกเทคนิค anti-DPI ที่ผิดปกติจาก AutoConnector คุณจะต้องเปิด AutoConnector บนโฮสต์โดยหลีกเลี่ยง (ข้อยกเว้น) แอนตี้ไวรัสและไฟร์วอลล์ หรือปิดมันทั้งหมดชั่วคราว แล้วรีบูตคอมพิวเตอร์ ใช่ แม้จะดูตลกที่ต้องแนะนำ แต่ให้รีบูตคอมพิวเตอร์ เพราะโปรแกรม VPN มักมีปัญหาและอาจทิ้งอุปกรณ์ TUN ไว้บนโฮสต์ในสภาพกึ่งทำงาน หลังรีบูตอย่าเพิ่งเปิด VPN ให้ทดสอบ AutoConnector ก่อน

  5. ลองหลบหลีกการบล็อก ปุ่มเปิดอยู่ในการตั้งค่าหรือบนหน้าจอหลัก (มองหาปุ่มที่อยู่ทางขวาของวงกลมใหญ่สีเทา/เขียว) คุณจะต้องใช้เวลาราว ~15 นาที ซึ่งประกอบด้วย 3 กลุ่ม:
	  - เอนจินพร็อกซี ลองโหมด Coalescing* ใดก็ได้ เป็นไปได้สูงว่าจะทำงานทันที แต่ในขณะเดียวกันรูปภาพ/วิดีโอใน Telegram จะไม่โหลด (นี่ไม่ใช่บั๊ก แต่เป็นฟีเจอร์/การประนีประนอม) จากนั้นลองโหมด Split* หากทำงานได้ — รูปภาพจะโหลด หรือคืนค่าเป็น "ปิดใช้งาน"
	  - ลอง "การแข่งขันของ upstream แบบขนาน" หมายความว่าเมื่อ Telegram ทำการเชื่อมต่อ 1 ครั้งไปยังพร็อกซี AutoConnector จะทำการเชื่อมต่อ 5-30 ครั้งไปยังพร็อกซี MTProto ต่าง ๆ และเสียบตัวที่ดีที่สุดให้ Telegram ในการตั้งค่าแอปสามารถเลือกไทม์เอาต์ (3-5 วินาที) และจำนวน upstream แบบขนานได้ถึง 30 ตัว
	  - เปิด "ไล่ลองเทคนิค anti-DPI อัตโนมัติ" โปรแกรมจะไล่ลองมันเอง
	  - เพื่อให้ Telegram สลับได้เร็วขึ้น ในการตั้งค่าพร็อกซี (ใน Telegram) ให้สลับด้วยมือทุก 5-10 วินาทีไปยังพอร์ตถัดไปวนเป็นวงกลม 55001->55002->55001->...

6. ชุดการตั้งค่าที่ทรงพลัง/เร็วที่สุด:
